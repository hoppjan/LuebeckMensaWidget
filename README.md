# LübeckMensaWidget

Android Widget for the Menu of Mensa Lübeck

## Build debug version

1. Clone the project
2. go into the repo
3. execute `./gradlew app:assembleDebug`

If you encounter any problems, fix them yourself (or open an issue)!
